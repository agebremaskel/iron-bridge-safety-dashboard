# Iron Bridge OPF Safety Dashboard

Static GitHub Pages package for the Iron Bridge OPF Safety Dashboard.

## Important

This package contains internal health and safety information and personnel names. Use a private/internal GitHub repository and restrict Pages access in line with Fortescue requirements.

## Publish Through GitHub Pages

1. Create or choose a private GitHub repository.
2. Copy the contents of this `github-pages` folder into the repository root.
3. Commit and push the files.
4. In GitHub, open Settings > Pages.
5. Set the source to deploy from the main branch root.
6. Share the generated Pages URL only with approved viewers.

The dashboard data is split into `data/datasets/*.txt` files so the site stays within GitHub file size limits.
